Storage Tips

  • SSD storage on pi1 and pi4 is available at /mnt/ssd
  • Create a custom StorageClass with pathPattern: /mnt/ssd/... to use SSDs
  • The default local-path StorageClass uses the root filesystem (SD card)
  • Use nodeAffinity to pin storage-heavy workloads to SSD-equipped nodes
