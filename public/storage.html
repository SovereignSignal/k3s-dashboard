Storage Tips
  • SSD storage on pi1 and pi4 is available via the local-path-ssd StorageClass
  • The default local-path StorageClass uses the root filesystem (SD card)
  • Use storageClassName: local-path-ssd in your PVC to use SSD storage
  • PVCs with SSD storage will only be scheduled on SSD-equipped nodes
